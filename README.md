# openclawskills.xyz landing

Static HTML + Tailwind CDN build.

## Deploy to Vercel

**Option 1 — Vercel CLI**
```bash
cd /root/clawd/builds/openclawskills-landing
vercel --prod
```

**Option 2 — Vercel Dashboard**
1. Create a new project in Vercel.
2. Upload the contents of this folder (or link the repo).
3. Framework preset: **Other** (static).
4. Build command: **None**
5. Output directory: **.**
